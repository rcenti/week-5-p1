Exercise 1.3

Your sketch:

Instructions

Create a "tail" for the mouse, as shown in the video below. The tail is created by saving the 5 most recent mouse coordinates in arrays then using those array to draw circles at the saved coordinates. The circle at the most recent coordinates should be 20px in diameter. The circles should decrease in size by 2px from the second most recent coordinate to the oldest coordinate.

Aim to separate the main tasks into functions for readability.

When you are done, check your work using check-exercise1-3.html.

Expected output:

You will need two empty global array variables. One will store the mouse's x coordinates and one will store the y coordinates.

There is no need to implement a mouse event function for this exercise. In draw(), add mouseX and mouseY to the appropriate arrays. As the tail is created using the five most recent mouse locations, use an if statement to check if the arrays contain more than 5 elements. If so, remove the oldest. There are two main ways to achieve this: you can either add new coordinates to the end of the arrays and remove the first element, or you can add new elements at the beginning of the arrays and remove the last element. Look at the documentation for the array methods linked below.

Use a for loop to draw circles at the saved coordinates. You will need to do a little maths to figure out the diameter for each circle. Try to write this maths as a calculation using the index of the coordinate in the loop. If you add new coordinates to the end of each array:

  • The circle at the coordinates stored at arrayName.length - 1 should be 20px in diameter.
  • The circle at the coordinates stored at arrayName.length - 2 should be 18px in diameter.
  • The circle at the coordinates stored at arrayName.length - 3 should be 16px in diameter.
  • ...and so on...

Some useful documentation:

  • arrayName.pop()
  • arrayName.shift()
  • arrayName.push()
  • arrayName.unshift()
