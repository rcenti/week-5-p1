Exercise 1.1

Your sketch:

Instructions

Add setup() and draw() to exercise1-1/sketch.js and create a 400px by 400px canvas.

Create two arrays containing the values given in the table below:

diameters colourVals
360 60
280 140
200 180
150 255

Use the arrays to draw concentric circles with different fill colours in the middle of your canvas. You can use the colourVals array to set the value of one or more colour channels.

When the user clicks their mouse, remove the last element of the colourVals array and insert the removed value at the beginning of the array. For example, the first time the mouse is clicked, the last element (255) will be removed and inserted at the beginning so the colourVals array becomes [ 255, 60, 140, 180].

When you are done, check your work using check-exercise1-1.html.

Expected output:

Both arrays should be saved as global variables. Because you know the values that will be stored in the arrays, they can be populated at the same time as they are created. See the lecture slides for an example.

Although it is possible to draw the circles without using a loop, you should ideally use a for loop to iterate through the arrays. The example / template in the lecture slides shows exactly how to set up the for loop. You can copy the code from the slides but make sure you understand it as loops are a very important programming concept. If you're at all unsure, please ask for help.

Before drawing each circle, use fill() to set the fill colour. At least one of the colour channels should use an element from the colourVals array. All circles should be drawn at the same location and use the diameters array to set the width.

Finally, implement the mouseClicked() event function. The lecture slides show two ways to remove elements from an array. One of those two methods does exactly what you need! Save the removed element as a variable then add the variable back to the beginning of the array. The lecture slides show three ways to add elements to an array. Choose the most appropriate approach for the task.
