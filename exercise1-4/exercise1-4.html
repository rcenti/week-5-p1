Instructions

A common use case for arrays is to make it possible to allow users to undo changes they have made. For example, Google Docs and Photoshop both have undo functionality and your web browser's back button is another type of undo functionality.

Undo functionality can be created by maintaining a history of user actions. To create the history, each user action is pushed to the end of an array. When the user wants to undo their action, the last element is popped from the array. Side note: a data structure that adds elements to the end and removes elements in reverse order is known as a "stack".

In this exercise, you will create a (very) simple drawing application that allows the user to undo their actions. Create a sketch of any size. When the user clicks the mouse add a new point, ellipse, or circle to the canvas at the mouse coordinates. When the user presses "z", remove the most recent shape.

When you are done, check your work using check-exercise1-4.html.

Expected output:

You will need two empty global array variables. One will store the x coordinates and one will store the y coordinates.

Implement the mouseClicked() event function. When the mouse is clicked, push the mouse x and y coordinates to the end of the appropriate arrays.

Implement the keyPressed() event function. When key is "z", pop the last element in each array.

Use the arrays to draw shapes (points, circles, or ellipses) on the canvas at the click locations.
